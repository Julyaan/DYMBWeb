# DYMBWeb

提交项目更新步骤：
在右击项目文件夹，选择'git bash here'


```
git add .
git commit -m '更新内容简要概括'
git push (输入用户名和密码)
```

---
更新项目步骤：
在右击项目文件夹，选择'git bash here'


```
git pull(输入用户名和密码)
```

---
运行项目：
命令行进入'DYMBWeb'：

```
copy bin/www.example bin/www (第一次运行即可)
npm install supervisor -g（仅在第一次运行时输入）
npm install
npm start
```
浏览器访问'localhost:3000'
